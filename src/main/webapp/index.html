Este curso tiene una duración de 4 meses. Se divide en dos partes: Frontend y Backend.

La primera parte vimos como es la estructura básica de una página HTML, como se le puede dar estilos usando CSS y funcionalidad utilizando el lenguaje JAVASCRIPT. Tambien nos adentramos en framework de CSS, en especial bootstrap para crear páginas web responsive.

A su vez, usamos una herramienta creo que indispensable al dia de hoy que es GIT, el cual nos permite tener un control de version de nuestro proyecto como tambien un respaldo.

En la segunda parte, instalamos XAMPP para el manejo de base de datos MySQL. Como se crea una base de datos, sus tablas y como se insertan, actualizan y eliminan los registro.

Instalamos NetBeans para el manejo del lenguaje JAVA. Aprendimo a como conectarnos a la base de datos y como hacer un CRUD en ella desde el lenguaje JAVA
